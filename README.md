#Скрипт настройки безопасности сервера

## 🚀 Как использовать?

### 1. Скопировать и вставить этот код в терминале на сервере VPS/VDS
#### это выполняется из под root
```bash
rm -rf ~/script-install-3x-ui-and-signed-ssl-cert && git clone https://github.com/SibMan54/script-install-3x-ui-and-signed-ssl-cert.git && cd script-install-3x-ui-and-signed-ssl-cert && chmod +x script_install_3x-ui_and_signed_cert.sh && ./script_install_3x-ui_and_signed_cert.sh
```
#### это выполняетя из под обычного пользователя
```bash
rm -rf ~/script-install-3x-ui-and-signed-ssl-cert && git clone https://github.com/SibMan54/script-install-3x-ui-and-signed-ssl-cert.git && cd script-install-3x-ui-and-signed-ssl-cert && sudo chmod +x script_install_3x-ui_and_signed_cert.sh && sudo ./script_install_3x-ui_and_signed_cert.sh
```
