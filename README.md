# Скрипт настройки безопасности сервера

## 🚀 Как использовать?

### Скопировать и вставить этот код в терминале на сервере VPS/VDS
#### это выполняется из под root
```bash
rm -rf ~/server-security-configuration-script && git clone https://github.com/SibMan54/server-security-configuration-script.git && cd server-security-configuration-script && chmod +x security_configuration_script.sh && ./security_configuration_script.sh && rm -rf ~/server-security-configuration-script
```
#### это выполняетя из под обычного пользователя
```bash
rm -rf ~/server-security-configuration-script && git clone https://github.com/SibMan54/server-security-configuration-script.git && cd server-security-configuration-script && sudo chmod +x security_configuration_script.sh && sudo ./security_configuration_script.sh && rm -rf ~/server-security-configuration-script
```
